Skippysunday.net

Description: This project is the website you are looking at! Made with HTML 5, CSS 3, Javascript ES6, hosted on an Apache2 webserver, and DNS registered through Google Domains, it's my little corner of the internet to share what I have working on.

Technical Details: This website is written entirely by me, and I learned everything in my own time. I started out with just a few HTML files, but it quickly grew as I added more about myself. Now, it includes 5 HTML files (one of which contains some PHP), 7 CSS files, and 5 Javascript files, cumulating to a total of over 1000 lines of code.

Of course, HTML and CSS are not technically programming languages, rather markup languages. Nevertheless, I still consider it within the same vein, and completing a proper web dev project is still a big step in a larger computer science dream.

As far as hosting goes, I have a free tier account with Oracle Cloud Infrastructure, and run a 4 OCPU 24 GB of RAM server, and Ubuntu 20.04. This server has been an excellent practice in not only using alternative operating systems to Windows, but also as a practice in system administration. Everything you see has been served by an Apache 2 webserver

As for my domain, it is registered with Google Domains. It is also SSL certified, meaning your connection to this website is secured through HTTPS!
